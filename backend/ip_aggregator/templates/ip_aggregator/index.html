IP Aggregator API

Aggregate IPv4 and IPv6 networks, compute derived statistics, and export the data in the format that best fits your workflow. Use this page as a quick reference for working with the API endpoints that power the networking tooling in this portfolio.

Available Endpoints

  • POST /ip_aggregator/api/ipaggregator/ — Submit a JSON payload with the IP networks you would like to aggregate.
  • GET /ip_aggregator/api/ipaddresses/?ip_addresses=... — Provide a comma separated list of CIDR blocks and optional query parameters to instantly summarise networks.

Sample Request

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{
        "ip_addresses": ["10.0.0.0/24", "10.0.1.0/24", "2001:db8::/64"],
        "output_format": "cidr",
        "why_blocked": "Abuse complaints"
      }' \
  http://localhost:8000/ip_aggregator/api/ipaggregator/

The response contains pre-computed summaries, aggregated ranges, and any additional metadata you request, allowing you to plug the results straight into spreadsheets, monitoring dashboards, or incident reports.
